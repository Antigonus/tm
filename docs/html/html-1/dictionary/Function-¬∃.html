Function ¬∃

Package:

TM

Syntax: 

  ¬∃ TM PRED &OPTIONAL CONT-TRUE CONT-FALSE → result

Arguments and Values:

  • TM : a T
  • PRED : a T
  • CONT-TRUE : a T
  • CONT-FALSE : a T
  • result : a T.

Description:

When true, there does not exist an object where pred holds, and tm is at rightmost. When false, tm is on an cell with an object where pred holds.
