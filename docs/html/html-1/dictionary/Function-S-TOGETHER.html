Function s-together

Package:

TM

Syntax: 

  s-together TMS &OPTIONAL CONT-OK CONT-EXISTS-ON-RIGHTMOST → result

Arguments and Values:

  • TMS : a T
  • CONT-OK : a T
  • CONT-EXISTS-ON-RIGHTMOST : a T
  • result : a T.

Description:

s-together is passed a list of tape machines. Each time s-together is called, all machines in the list are stepped. If any of the machines can not be stepped, then none of the machines are stepped. cont-exists-end is called with an iterator on the first of the tms that could not be stepped.
