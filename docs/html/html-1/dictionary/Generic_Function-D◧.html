Generic Function d◧

Package:

TM

Syntax: 

  
      
          d◧
      
   
              TM::TM
          
          
              "&OPTIONAL"
          
          
              TM::SPILL
          
          
              TM::CONT-OK
          
          
              TM::CONT-RIGHTMOST
          
          
              TM::CONT-NO-ALLOC
          
    → result

Arguments and Values:

  • TM : a T
  • SPILL : a T
  • CONT-OK : a T
  • CONT-RIGHTMOST : a T
  • CONT-NO-ALLOC : a T
  • result : a T.

Description:

Similar to #'d but the leftmost cell is deallocated independent of where the head is located, unless the leftmost cell is the rightmost cell, in which case cont-rightmost is called. If the tape head is on the leftmost cell, it is moved to the new leftmost cell.
