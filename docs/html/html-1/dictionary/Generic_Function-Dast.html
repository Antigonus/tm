Generic Function d*

Package:

TM

Syntax: 

  
      
          d*
      
   
              TM::TM
          
          
              "&OPTIONAL"
          
          
              TM::SPILL
          
          
              TM::CONT-RIGHTMOST
          
          
              TM::CONT-NO-ALLOC
          
    → result

Arguments and Values:

  • TM : a T
  • SPILL : a T
  • CONT-RIGHTMOST : a T
  • CONT-NO-ALLOC : a T
  • result : a T.

Description:

Deallocates all cells right of the head up to and including rightmost. If spill is not ∅, then the deallocated right hand side is moved to it. If spill is not ∅, and cells can not be moved to it, the objects are moved to spill via #'a. It is only in this last case that we might end up taking the cont-no-alloc exit.
