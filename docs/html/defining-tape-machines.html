Doing Work

Here is a simple program that creates a fresh list by squaring the items in a source list. 


             (let(
                  (source (mount {1 2 3}))
                  (destination (tm-mk 'tm-list))
                  )
               (⟳ (λ(cont-loop cont-return)
                    (as destination (expt (r source) 2))
                    (s source cont-loop cont-return)
                    ))
               (unmount destination)
             )
When running this it produces: 

              (LIST 1 4 9)

Even when following a regime of non-destructive programming, it still makes sense to reuse state counters, such as the counter in a do loop. A tape machine should be treated as such a state variable.
