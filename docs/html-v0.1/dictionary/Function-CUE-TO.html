Function cue-to

Package:

TM

Syntax: 

  cue-to TM-CUED TM-ORIG → result

Arguments and Values:

  • TM-CUED : a T
  • TM-ORIG : a T
  • result : a T.

Description:

Unless tm-cued type is (typep (type-of tm-orig)), tm-cued is changed to the type of tm-orig. tm-cued is either already on the same tape as tm-orig, or it mounts it. tm-cued is cued to the same cell that tm-orig's head is on. Returns tm-cued.
