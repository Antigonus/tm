Function ⟳

Package:

TM

Syntax: 

  ⟳ WORK → result

Arguments and Values:

  • WORK : a T
  • result : a T.

Description:

⟳ (pronounced "do") accepts a work function. This work function is to take a single step, whatever such a step may be. The work function accepts two continuations. Typically these are called 'cont-loop', and 'cont-return'. When the work function continues with cont-loop, it is immediately called again. When it continues with cont-return, ⟳ returns.
